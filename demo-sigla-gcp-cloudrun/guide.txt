0- Autenticarsi su GCP con il seguente comando:

> gcloud auth login

1- Impostare l'ID del progetto associato alla cloud shell:

> gcloud config set project [project_id]
(esempio: gcloud config set project demo-sigla)

2- Impostare l'ID del progetto come variabile d'ambiente:
> export project_id=[project_id]

3- Assicurarsi che tutti gli script nella directory siano eseguibili:
> chmod +x *.sh

4- Eseguire il deploy della soluzione:

> ./start-demo.sh
